# tm-build

#### 介绍
> 通用webpack配置封装，保留webpack api 一致
- 支持vue
- 适用场景: appScene WMS（中后台项目）、H5、""(空则为普通开发插件打包)

#### 基本使用
```
// 全局安装
npm i -g tm-build

// 项目必要结构
|--src
|   |-main.js // 默认入口（可通过tm.config.js配置修改）
|
|-- tm.config.js // 可自定义配置webpack

```
